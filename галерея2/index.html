Картинная галерея, серия вторая

Примечание

Вы можете использовать стрелочки, чтобы перемещаться между изображениями в режиме показа плашек. Чтобы выйти из показа плашек, нужно нажать мышкой вне плашки, чтобы переключиться с плашек обратно на сетку.
В этой галереи нет изображений, которые могут оказаться неприемлемыми для зрителей.
Для скачивания воспользуйтесь wget или Down Them All! (пункт Ссылки) и скачайте файлы 1-51.
Большинство картин выполнено с помощью художественной техники родового этно-мазка новой эры.
Детализация картин различна, тематика также разнообразна. Некоторые картины выглядят так, словно они сотворены ребёнком, другие поставят в тупик даже бывалого зрителя.
Для рисования была использована программа GIMP — свободный графический редактор. А для векторной графики — Inkscape.
